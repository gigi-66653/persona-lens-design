

## Persona 卡片 — 奢侈品杂志风格升级

### 设计理念

借鉴高端时尚杂志（如 Vogue、Monocle）的排版美学，核心特征：

- **大量留白**：信息之间呼吸感更强，padding 加大
- **衬线感字体**：Persona Name 使用 serif 字体（Playfair Display），营造editorial质感
- **极细线条分隔**：用 0.5px 半透明线条代替粗分隔
- **字母间距拉宽**：标签和小字用大写 + 超宽 letter-spacing，模拟杂志排版
- **克制的配色**：减少蓝色饱和度，偏向冷灰蓝，更接近纸张质感
- **微妙的渐变光晕**：卡片背景保留毛玻璃但更内敛，阴影更柔和更扩散
- **黄金比例层级**：Name 特大号，描述适中，其余信息极小且克制

### 视觉对比（当前 → 升级后）

```text
当前                          升级后
─────────────────────────────────────────
Sans-serif 粗体标题      →   Serif 字体，优雅感
紧凑间距                  →   大留白，呼吸感
彩色 pill 标签            →   大写细字 + 宽间距
实线分隔                  →   极细半透明线
蓝色调阴影               →   中性冷灰柔和阴影
```

### 技术实现

1. **引入 Google Font `Playfair Display`**：在 `index.html` 添加字体链接，`tailwind.config.ts` 添加 `fontFamily.serif`
2. **更新 PersonaCard 组件**：
   - Name 使用 serif 字体，字号增大至 2xl，font-weight 降至 semibold
   - 描述文字行高加大，字色更浅
   - 核心维度标签改为大写无背景文字，用 `·` 分隔，letter-spacing 加宽
   - 品牌关系区域用更克制的排版，图标更小更淡
   - Posts 数量用大写微字
   - 整体 padding 从 p-6 增至 p-8
3. **更新 CSS 变量**：降低主色饱和度，阴影改为中性灰调
4. **展开区域**：详情卡片改为无边框、纯文字排版风格
